Make sure you have ffmpeg installed (if you don't have it see how in the end)

- Copy the path where the .m4s files are
- Run steam_video_converter.bat
- Paste the path
- The program will output a video.mp4 file in the same directory where steam_video_converter.bat is



Install ffmpeg:

- Download ffmpeg for windows
- Choose somewhere to store it
- Open the bin file and copy its path

- Win key > Edit the system environment variables > Environment Variables... > System variables > Path > New > paste the copied path > Ok > Ok > Ok
